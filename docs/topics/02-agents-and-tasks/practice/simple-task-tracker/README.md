# 实践项目：简单任务追踪

## 目标

学习使用 Claude Code 的 Task 工具来管理和追踪开发任务。

## 背景

你正在开发一个小功能，需要完成以下几个步骤：
1. 阅读相关代码
2. 编写测试
3. 实现功能
4. 运行测试
5. 清理代码

## 任务

使用 Claude Code 完成以下任务，并观察 Task 工具的使用：

### 1. 创建任务列表

告诉 Claude Code：
```
"创建一个任务列表，包含以下步骤：
- 阅读 src/utils.py 文件
- 为 add 函数编写测试
- 运行测试"
```

### 2. 执行任务

按照任务列表逐步执行，观察 Claude Code 如何更新任务状态。

### 3. 观察状态变化

注意任务状态的变化：
- `pending` - 待处理
- `in_progress` - 进行中
- `completed` - 已完成

## 学习要点

- TaskCreate：创建新任务
- TaskUpdate：更新任务状态
- TaskList：查看所有任务
- TaskGet：获取任务详情

## 预期结果

你应该能够：
1. 理解如何创建任务
2. 看到任务状态的变化
3. 理解任务之间的依赖关系（blocks/blockedBy）

## 进阶挑战

尝试创建有依赖关系的任务：
- 任务 2 必须在任务 1 完成后才能开始
